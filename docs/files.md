# 📁 File & Folder Management

Essential commands for handling files and directories.

## 📂 Navigation

```bash
pwd
ls -lah
cd ../
```

## 🗂️ File Operations

```bash
touch newfile.txt
cat file.txt
head -n 10 file.txt
tail -f logfile.log
```

## 📁 Folder Operations

```bash
mkdir new-folder
rm -rf old-folder
cp -r src/ dest/
mv oldname.txt newname.txt
```
